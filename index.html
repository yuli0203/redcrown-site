Enzymatic Lab Educational Model

Mixed Reality | Physics-Based Hand Interactions

This complementary Meta Quest 3 model is used to explain theoretical material in greater depth as part of an enzymatic laboratory experiment at the Technion.

The model can be overlaid on any existing real table, allowing students to interact with virtual scientific content while maintaining a clear connection to their physical environment.

We implemented physics-based hand interactions that allow students to pick up molecules, rotate and zoom in on scientific models, and perform various interactive actions. These visualizations and interactions help students in the Wolfson Faculty of Chemical Engineering at the Technion better understand complex concepts related to enzymatic reactions and polarized light.

Immersive Fugacity VR Experiment

Virtual Reality | Hand Gestures | Controllers

This fully immersive Meta Quest 3 VR experiment is used to teach fugacity as part of the Thermodynamics B course at the Technion.

The experiment is approximately one hour long and is designed to be performed by multiple student groups simultaneously (up to 12 devices). Students are also trained in laboratory safety and fire safety procedures as part of the experience.

The project includes a virtual instructor who guides students through the full experiment using spoken instructions. The Meta Quest 3 headsets are paired with tablets, allowing lab instructors to monitor each group’s progress and receive data files generated during the experiment.

Molecular Scientific ML Research Tool

PC | Machine-Learning | Python Server

LiveMol is a research platform for molecular modeling, built for the Wolfson Faculty of Chemical Engineering at the Technion. It combines real-time 3D visualization with scientifically accurate computational chemistry.

Researchers build and edit molecules directly in 3D, then run quantum chemistry calculations on the structure in front of them and watch the results return onto the model. Supported calculations include single point energy, geometry and live optimization, vibrational frequency analysis, and transition state optimization.

It runs as a Unity client driving a Python compute server, which keeps the interface responsive while heavy calculations run separately. The scientific work is handled by established engines including ASE, tblite, Open Babel, and Sella, with machine-learning components planned alongside them.

This project is currently in development, first for PC, later stage for AR.

Physical Meta Quest Classroom Setup

Multi-User VR | Meta Quest 3 Deployment | Instructor Monitoring

An end-to-end physical VR classroom at the Technion, built to reliably support multiple student groups using immersive educational applications at the same time.

We consulted across the whole implementation, from planning and hardware selection to classroom layout, device configuration, deployment, testing, and instructor workflows, all shaped around the institution's available space, teaching requirements, safety, and technical infrastructure.

The room runs up to 12 Meta Quest 3 headsets simultaneously, with dedicated seating, spacing, and charging and storage, plus a paired tablet for each headset so instructors can monitor group progress, support students, and collect the data files generated in each session.
